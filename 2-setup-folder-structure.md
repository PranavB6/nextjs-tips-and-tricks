# Folder Structure

```
src/
├─ api/        -> contains the API to connect to timeey-backend
├─ app/        -> routes
├─ components/ -> components that are shared across routes
├─ hooks/      -> react hooks
├─ models/     -> types and interfaces to be used by our app
├─ store/      -> react contexts
├─ utils/      -> helper functions
```
